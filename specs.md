# 주간 시간표 웹앱 기술 명세서 (specs.md)

## 1. 주요 기능 (Functionality)

### 1.1 UI/UX 고도화
- **GSAP Draggable 도입**: 부드러운 드래그 구현 및 투명도 현상 제거.
- **정밀 그리드**: 1시간/10분 단위 그리드 시각화.
- **상/하단 리사이즈(Resizing)**: 일정 박스의 위쪽 혹은 아래쪽 가장자리를 드래그하여 시작 시간과 종료 시간을 10분 단위로 정절할 수 있습니다.

### 1.2 드래그 앤 드롭 (GSAP 기반)
- **가로 중앙 기준 드롭 판정**: 항목의 가로 중앙 좌표를 기준으로 요일 스냅.
- **이동과 리사이즈 분리**: `cancel: ".resize-handle"` 옵션을 통해 리사이즈 핸들 조작 시에는 전체 이동이 발생하지 않도록 분리했습니다.
- **소요 시간 보존**: 전체 이동 시에는 길이를 유지, 리사이즈 시에는 길이를 변경.

## 3. 주요 기술 세부사항 (Technical Details)

### 3.1 핵심 로직
- **`handleResizeStart`**: 
    - 마우스 이동량에 따라 실시간으로 `startTime` 또는 `endTime` 데이터를 업데이트합니다.
    - 시작 시간이 종료 시간보다 늦어지거나, 종료 시간이 시작 시간보다 빨라지지 않도록 최소 10분의 간격을 강제합니다.
    - 10분 단위 스냅 로직 적용.

## 4. 스타일링 규칙 (CSS Rules)
- **`.resize-handle`**: 일정 박스 상/하단에 위치한 투명 핸들. `cursor: ns-resize` 적용.
